## ğŸš€ Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)
- Constelia API Key

### Installation

1. Clone the repository
```bash
git clone https://github.com/luinbytes/fantasy.comet.git
cd fantasy.comet
```

2. Install dependencies
```bash
npm install
```

### Development

Run the app in development mode:
```bash
npm run dev:app
```

### Building

Build the app:
```bash
npm run build
```

Start the built app:
```bash
npm start
```
