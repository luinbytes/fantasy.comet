### Prerequisites

- Node.js (v14 or higher)

### Installation

1. Clone the repository
```bash
git clone https://github.com/luinbytes/fantasy.comet.git
cd fantasy.comet
```

2. Install dependencies
```bash
npm install
```

### Development

Run the app in development mode:

Build the app:
```bash
npm run build
```

